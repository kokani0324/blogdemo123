## Vue 怎麼放(免安裝)
1. 把 `vue.global.js` 複製到 `src/main/resources/static/js/`
2. 每個頁面 = 一個 `.html` + 一個 `.js`,例如 `blogs.html` 配 `blogs.js`
3. HTML 裡:`<script src="/js/vue.global.js"></script>` 再 `<script src="/js/blogs.js"></script>`
4. `.js` 裡 `Vue.createApp({ data(){...}, methods:{ getBlogs(){ fetch('/blogs').then(...) } } }).mount('#app')`
5. 啟動後直接開 `http://localhost:8080/blogs.html`(同一個 Spring Boot 服務,沒有跨域問題)

## 風險與建議
1. **Vue 是免安裝版**(`vue.global.js`),不用碰 npm/Vite,難度比想像低很多;`v-for`/`v-model` 會讓列表和表單更快。
2. **圖片上傳**對新手最容易卡(Day 2、4),卡住先用 base64 最簡單。
3. **進度排序=優先序**:部落格(主責)第一週+Day 6-7 做完,核心交付最先到手。
4. **體驗活動只做基礎後端**,不要陷進場次/報名;自己留小後端,才不會被隊友 API 卡住。
5. **產地地圖**需要農場經緯度,Day 9 前先確認資料庫有沒有座標。
6. 週末當緩衝,每天先達「完成標準」再往下,不要累積半成品。

## 每天遇到問題的 SOP
1. 先看是哪一層出錯(Controller / Service / Dao / SQL / 前端 fetch)
2. 對照 springboot-mall(後端)或 vue_class(前端)同樣功能怎麼寫
3. 還是卡 → 把錯誤訊息丟給 Claude
